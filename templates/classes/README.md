# Classes

Use this folder for lecture notes, converted PDFs, course indexes, and study guides.

## Workflow

1. Put raw lecture files in `_inbox/`.
2. Convert them to Markdown in `_converted/`.
3. Render PDF pages into `_pages/` for diagrams and visuals.
4. Create one course index note per course.
5. Ask Claude to read the converted Markdown first and rendered page images when needed.

## Folders

- `_inbox/`: raw PDFs, PPTX, DOCX
- `_converted/`: Markdown generated by MarkItDown
- `_pages/`: PDF pages rendered as PNG images for visual inspection

## Claude Prompt

```text
Use the lecture-ingest skill. Convert the lecture files in ~/dev-vault/classes/_inbox to Markdown, store them in ~/dev-vault/classes/_converted, then create a course index with key concepts and source links.
```

Render PDF page images:

```bash
~/dev-vault/classes/render-pages.sh
```
